SvgCharts4Xsl

SvgCharts4Xsl is a set of utility XSL templates to generate SVG charts. It can be used to embed dynamic charts in HTML and XSL-FO.

Usage

Import the respective chart XSL file in your stylesheet and call the appropriate method.

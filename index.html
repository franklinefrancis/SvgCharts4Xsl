SvgCharts4Xsl by franklinefrancis

Utility XSL templates to generate SVG charts. Can be used to embed dynamic charts in HTML and XSL-FO.

Utility XSL templates to generate SVG charts. Can be used to embed dynamic charts in HTML and XSL-FO.

Dependencies

Piechart requires the transformer to support EXSLT extensions

Install

Import the respective chart XSL file in your stylesheet and call the appropriate method.
